User information

NB: Do not use slash '/', whitespace ' ' or '_' in user or project names (e.g. MyName, MyProject not 'My Name', 'My Project', 'My_Project'). You should also use a descriptive project name (instead of 'test' use e.g. 'dataset-name-run-1'). You can use the same project name over and over but if you're then using the same classifiers the first run will not get overridden (but new ones will be appended to the project).
